# ESP8266-I2C-Driver
Bug fixes of native Arduino ESP8266 core I2C Driver

Copy and relpace "twi.h", "core_esp8266_si2c.c" in folder %USERPROFILE%\AppData\Local\Arduino15\packages\esp8266\hardware\esp8266\2.4.0\cores\esp8266

Copy and replace "Wire.h", "Wire.cpp" in folder %USERPROFILE%\AppData\Local\Arduino15\packages\esp8266\hardware\esp8266\2.4.0\libraries\Wire

Enjoy.
